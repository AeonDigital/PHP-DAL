Instalação
----------

Instale em seu projeto usando o composer:


**Via terminal**

.. code-block:: console

    composer require aeondigital/phpdal


**Via composer.json**

.. code-block:: json

    "require": {
        "aeondigital/phpdal": "dev-master"
    }
